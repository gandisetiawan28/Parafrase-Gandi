# Walkthrough Final: Parafrase Gandi Word Add-in

Proyek ini telah selesai dikembangkan dan siap untuk lingkungan produksi.

## Fitur Utama yang Diimplementasikan
1.  **Sistem Lisensi Aman**: Terhubung ke Google Sheets dengan penguncian 1-Perangkat-1-Kunci.
2.  **Mesin Gemini AI**: Mendukung model terbaru (2.5 Flash) dengan 3 variasi pilihan.
3.  **Retensi Struktur**: AI secara cerdas mempertahankan jumlah paragraf asli menggunakan tag `<p>`.
4.  **Otomatis Italic**: Istilah asing dalam teks Indonesia otomatis dicetak miring.
5.  **Multi-Language**: Pilihan output dalam Bahasa Indonesia, Inggris, Arab, dan Jepang.
6.  **WhatsApp Integrasi**: Tombol dukungan pelanggan langsung ke nomor Gandi.

## Teknis Deployment
- **Hosting**: GitHub Pages (via `npm run deploy`).
- **Update**: Otomatis tersinkronisasi ke pelanggan setiap ada perubahan kode.
- **Manifest**: Menggunakan URL produksi `https://gandisetiawan28.github.io/Parafrase-Gandi/`.
